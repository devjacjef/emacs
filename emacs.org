* Packages

Below is the setup for using packages in GNU Emacs. Following the setup are all the packages I have chosen to use for emacs.

#+BEGIN_SRC emacs-lisp
  (require 'package)
  (add-to-list 'package-archives '("gnu"   . "https://elpa.gnu.org/packages/"))
  (add-to-list 'package-archives '("melpa" . "https://melpa.org/packages/"))
  (package-initialize)

  (unless (package-installed-p 'use-package)
    (package-refresh-contents)
    (package-install 'use-package))
#+END_SRC

*** LSP Mode

In =:config= I set =lsp-eldoc-render-all= to render all the information the LSP gives us and just to be explicit I enable =lsp-eldoc-enable-hover=.

#+BEGIN_SRC emacs-lisp
  (use-package lsp-mode
    :ensure t
    :hook (
	 (go-mode . lsp)) ;; Enable LSP for Go. Tends to be buggy...
    :config 
	   (setq lsp-eldoc-render-all t)
	   (setq lsp-eldoc-enable-hover t))
#+END_SRC

** Eldoc

When using =lsp-mode=, enable =eldoc-box-hover-at-point-mode= to allow for docs when hovering a function.

#+BEGIN_SRC emacs-lisp
  (use-package eldoc-box
    :hook (lsp-mode . eldoc-box-hover-at-point-mode))
#+END_SRC
